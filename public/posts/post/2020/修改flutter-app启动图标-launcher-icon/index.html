修改Flutter APP启动图标/Launcher Icon

Thursday, Dec 10, 2020 | 1 minute read | Updated at Thursday, Dec 10, 2020

开门见山

为了简化修改Flutter启动图标的过程, Flutter Launcher Icons诞生了.

开始

访问 Flutter Launcher Icons 库网址 . 访问慢的同学可以使用国内镜像 .

修改你的pubspec.yaml文件(版本号替换为最新的):

` dev_dependencies:   flutter_launcher_icons: ^0.8.1`

在资源目录中创建你的icon文件, 比如路径assets/icon/icon.png.

然后再次修改pubspec.yaml:

` flutter_icons:   android: "launcher_icon"   ios: true   image_path: "assets/icon/icon.png"`

在终端中分别运行以下命令:

`flutter pub get`

`flutter pub run flutter_launcher_icons:main`
